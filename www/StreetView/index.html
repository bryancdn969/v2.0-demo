StreetView class

    Create

    StreetView.getPanorama()
    Gets a new panorama instance.

    Methods

    setPosition()
    Sets the current LatLng position for the Street View panorama.
    setPov()
    Sets the point of view for the Street View panorama.
    getPosition()
    Return the current camera position (LatLng).
    getLocation()
    Return the current location, including links.
    getLinks()
    Return the link of next location
    setPanningGesturesEnabled()
    Toggles the ability for users to use pan around on the panorama using gestures.
    getPanningGesturesEnabled()
    Retrun true if the panning gesture is enabled.
    setZoomGesturesEnabled()
    Toggles the ability for users to zoom on the panorama using gestures.
    getZoomGesturesEnabled()
    Retrun true if the zooming gesture is enabled.
    setStreetNamesEnabled()
    Toggles the ability for users to see street names on the panorama.
    getStreetNamesEnabled()
    Retrun true if the street names control is enabled.
    setNavigationEnabled()
    Toggles the ability for users to move between panoramas.
    getNavigationEnabled()
    Retrun true if the navigation control is enabled.
    setVisible()
    Change visibility of the panorama.
    getVisible()
    Return true if the panorama is visible.
    remove()
    Remove the panorama.

    Events

    PANORAMA_READY
    Arguments: None
    This event is fired when the panorama is fully ready.
    PANORAMA_CAMERA_CHANGE
    Arguments: PanoramaCameraPosition
    This event is fired when the camra is moved (such as panning).
    PANORAMA_LOCATION_CHANGE
    Arguments: PanoramaLocation
    This event is fired when the camera location is moved.
    PANORAMA_CLICK
    Arguments: PanoramaOrientation
    This event is fired when you click on the panorama.
